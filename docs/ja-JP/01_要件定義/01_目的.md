# 目的

本プロジェクトでは、ロボットシミュレーション・実機・VR・外部アプリケーションを統合可能な Runtime 環境の構築を目的とする。

また、シミュレーション専用環境と実機専用環境を分離せず、同じ構造・同じデータを利用しながら段階的に実機へ移行できる環境を目指す。

さらに、単一用途のシミュレーターではなく、ロボット開発・検証・実験・遠隔操作までを継続的に扱える基盤を目指す。

## 実現したいこと

### シミュレーション構成

- ロボット構成の作成・再利用
- フィールド構成の作成・再利用
- Runtime 設定の切り替え
- アクチュエーター設定の管理
- センサー設定の管理
- シミュレーション条件の切り替え

### シミュレーション

- MuJoCo を利用した物理シミュレーション
- 実機特性を反映したシミュレーション
- 実機移行前検証
- HIL / SIL
- sysid

### シミュレーション連携

- 実機制御基板との連携
- Meridian 系通信との連携
- PCアプリケーション連携
- Python 連携（AI・制御研究）
- ログ・解析システム連携

### 外部連携

- VRシステムへのシミュレーション結果投影
- 外部 Runtime との同期
- 遠隔操作システム連携
- 外部センサー連携

## 最終的に目指す世界

シミュレーション・VR・実機・外部システムを分離せず、同じロボットを同じ構造のまま扱える Runtime 環境を実現する。

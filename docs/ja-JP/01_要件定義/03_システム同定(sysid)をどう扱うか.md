# システム同定(sysid)をどう扱うか

MuJoCo上のシミュレーション精度を向上するため、システム同定(sysid)機能を提供したい。  

本システムでは、利用者が手持ちのアクチェーターやロボットの実測データを元に sysid を実行し、その結果を MuJoCoデータセットに保存することで、シミュレーションへ反映できるようにする。

（以下校正中）

## sysid の対象

以下を sysid 対象とする。

- アクチェーター特性
  - PID応答
  - トルク特性
  - 速度特性
  - 制御遅延
  - デッドゾーン
  - 個体差

- ロボット組み立て状態
  - 関節摩擦
  - 重量バランス
  - 慣性変化
  - 配線や機構による負荷
  - 組み立て精度差

- シミュレーション環境
  - 接地摩擦
  - 床材特性
  - 外乱条件

## sysid の基本動作

本システムは、以下のような流れで sysid を行う。

入力
- 実機ログ
  - 目標角度
  - 実角度
  - 電流
  - IMU
  - 時刻情報
  - その他センサ情報

準備
- Runtime が sysid 用データセットを構築
- MuJoCo シミュレーションモデルへ適用
- 比較対象パラメータを選択

実行
- 実機ログと MuJoCo シミュレーション結果を比較
- 誤差が小さくなるようパラメータを調整
- 最適化結果を sysid 結果として出力

出力
- sysid パラメータ
- 比較ログ
- 評価結果
- Runtime 用設定データ

## sysid 結果の利用

sysid 結果は、MuJoCoデータセットへ組み込み可能とする。

これにより：

- 同一型番アクチェーターの共有
- 個体差比較
- 実機再現性向上
- 自作ロボットの特性再現

などを可能とする。
